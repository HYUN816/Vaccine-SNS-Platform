# 백신 후기 공유 SNS 플랫폼

## 개발 기간
- 2021-10-11 ~ 2021-10-15

## 기술 스택 
- Front-end : React  
- Back-end : Node.js, Express  
- DataBase : MongoDB, Mongoose  

## 기능 
- 회원가입, 로그인, 로그아웃, 회원 정보수정
- jwt토큰 로그인 유지 
- 게시판, 댓글 기능
